# Aplicacion_metodos_estadisticos_en_validacion_automatizada_y_auditoria_de_ordenes_de_trabajo_en_ENEL
Proyecto de investigación y análisis de datos enfocado en ENEL, donde se estudia información empresarial y del sector energético para identificar patrones, indicadores clave y oportunidades de mejora, apoyando la toma de decisiones mediante análisis estructurado y visual.
